三、制度
  1、会员静态收益
会员投资100元即可拥有一块自己的养殖牧场，并获得牧场收益。牧场收益固定，每天分润20，分润到160%出局，（牧场每人限购一块）
    1000元认养幼崽牦牛一头，每天按市场所有幼崽牦牛收益加权进行分润，根据实际牧场养殖经验，每只幼崽牦牛每天分润《2%~5%》），分润到160%出局，停止收益。每个会员最多认养5头幼崽牦牛。公司同时邮寄价值988元的产品包邮到家（公司每天限量认养）
    5000元认养成年牦牛一头，每天按市场所有黑牦牛收益加权进行分润（根据实际牧场养殖经验，每成年牦牛每天分润《3%~7%》），分润到160%出局，停止收益。每个会员每天最多认养3头成年牦牛。公司同时邮寄价值4988元的产品包邮到家（公司每天限量认养）
    10000元认养母牦牛一头，每天按市场所有母牦牛收益加权进行分润（根据实际牧场养殖经验，每只母牦牛每天分润《2%~10%》），分润到160%出局，停止收益。每个会员每天最多认养2头母牦牛。公司同时邮寄价值9888元的产品包邮到家（公司每天限量认养）
  
  2、会员动态收益
      1、推荐奖：推荐奖三级分销   一级6% 二级6% 三级6%（一次性到账）  // 购买商品的时候的

      2、回馈奖：1代4%  二代3%，2代3%  4到8代1% 每天分润后到账        // 下级收益的

      3、牛气奖（包含推荐奖和回馈将）：会员所有动态奖励到账后系统直接统计进入牛气奖，按：150%的比例到账（举例动态将100元，那么牛气将就是150元），每天释放。直推1人每天释放5%，直推5人每天释放10%，直推10人每天释放20%，直推超过10人，每增加一名直推，多释放2%，以此累计，直到每天释放100%。
3、商城：商城全部真实发货，购物币购物。
